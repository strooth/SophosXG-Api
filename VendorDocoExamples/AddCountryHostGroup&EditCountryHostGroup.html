Operation: Add Country Host Group / Edit Country Host Group
Description: To Add/Edit the Country Host Group. 

Sample Configuration



Parameter Mandatory Default Description
CountryHostNo  
Specify 'select'
CountryHost confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • NoComma
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
DescriptionNo  
Specify the Country Host Group description.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
NameNo  
Specify a name to identify the Country Host group.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • UTF-8 character(s) are allowed.
