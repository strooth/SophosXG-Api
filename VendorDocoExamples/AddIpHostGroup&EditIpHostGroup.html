Operation: Add Ip Host Group / Edit Ip Host Group
Description: To Add/Edit IP Host Group. 

Sample Configuration
<IPHostGroup> <Name>name</Name> <IPFamily>IPv4/IPv6</IPFamily><!-- default IPv4 --> <Description>Text</Description> <HostList> <Host>Hostname</Host> <Host>Hostname</Host> <Host>Hostname</Host> <Host>Hostname</Host> </HostList> </IPHostGroup>



Parameter Mandatory Default Description
NameNo  
Specify a name to identify the IP Host group.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • UTF-8 character(s) are allowed.
DescriptionNo  
Describe the IP Host Group.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
HostNo  
Select Host names to be added in the IP Host Group.
Host confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • NoComma
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
IPFamilyNo IPv4 
Select IP Family of the IP Host Group: IPv4 or IPv6.
IPFamily confines to:
  • Type is 'SCALAR'.
  • Only 'IPv4', 'IPv6' are allowed.
