Operation: Add Content Condition List / Update Content Condition List
Description: To Add Content Condition ListTo Edit Content Condition List 

Sample Configuration
<ContentConditionList> <Name>Name</Name> <Description>Description</Description> <Key>Key</Key> <ContentList> <ContentString>Content Text</ContentString> : </ContentList> </ContentConditionList>



Parameter Mandatory Default Description
NameYes  
Specify Content Condition List Name
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 255.
  • UTF-8 character(s) are allowed.
KeyYes  
Specify Content Condition List Key
Key confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 127.
  • UTF-8 character(s) are allowed.
DescriptionNo  
Specify Content Condition List Description.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 255.
Raw RegexesNo  
Specify Content Condition List Raw Regexes
Raw Regexes confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
  • Duplicate values will not be ignored.
