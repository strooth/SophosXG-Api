<PolicyRoute> <Name>Default</Name> <Description /> <IPFamily>{0/1}</IPFamily><!-- default IPv4 --> <Gateway>ip</Gateway> <Interface>{interface}</Interface> <DSCPMarking>0-Best Effort/1/2/3/4/5/6/7/8-Class 1(CS1)/9/10-Class 1,Gold(AF11)/11/12-Class1,Silver(AF12)/13/14-Class 1,Bronze(AF13)/15/16-Class 2(CS2)/17/18-Class 2,Gold(AF21)/19/20-Class 2,Silver(AF22)/21/22-Class 2,Bronze(AF23)/23/24-Class 3(CS3)/25/26-Class 3,Gold(AF31)/27/28-Class 3,Silver(AF32)/29/30-Class 3,Bronze(AF33)/31/32-Class 4(CS4)/33/34-Class 4,Gold(AF41)/35/36-Class 4,Silver(AF42)/37/38-Class 4,Bronze(AF43)/39/40-Class 5(CS5)/41/42/43/44/45/46-Expedited Forwarding(EF)/47/48-Control(CS6)/49/50/51/52/53/54/55/56-Control(CS7)/57/58/59/60/61/62/63</DSCPMarking> <SourceNetworks> <Network>Source Network</Network> <Network>Source Network</Network> : </SourceNetworks> <Services> <Service>servicename</Service> : </Services> <DestinationNetworks> <Network>Destinaiton Network</Network> <Network>Destinaiton Network</Network> : </DestinationNetworks> </PolicyRoute>



Parameter Mandatory Default Description
NameYes  
Specify a name for policy route.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 60.
  • UTF-8 character(s) are allowed.
DescriptionNo  
Specify a description for Policy Route.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 255.
IPFamilyYes IPv4 
Select the IP family.
IPFamily confines to:
  • Type is 'SCALAR'.
  • Only 'IPv4', 'IPv6' are allowed.
SourceNetworksNo  
Select the source network(s) of the packet to be routed.
SourceNetworks confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
DestinationNetworksNo  
Select the destination network(s) of the packet to be routed.
DestinationNetworks confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
ServicesNo  
Select the services(s) of the packet to be routed.
Services confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
DSCPMarkingNo  
Select the type of DSCP Marking to match the packets marked with the given DSCP value.
DSCPMarking confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Range 1 to 64 is allowed.
GatewayYes  
Select the gateway to which you want to forward the packet if the packet meets the configured matching criteria.
Gateway confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
InterfaceNo  
Select the incoming interface receiving the packet.
Interface confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
