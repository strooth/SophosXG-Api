Operation: Add Single OR Multiple Guest Users
Description: To Add Single/Multiple Guest Users 

Sample Configuration
<GuestUser> <Name>username</Name> <Email>emailid</Email> <UserValidity>Duration in Days</UserValidity> <ValidityStart>Immediately/AfterFirstLogin</ValidityStart> </GuestUser>



Parameter Mandatory Default Description
NoOfUsersYes  
Specify 'numberofusers'
NoOfUsers confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 1 to 100 is allowed.
  • Maximum digits allowed are 3.
NameYes  
Specify 'name'
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
UserValidityYes  
Specify 'genexpiryperiod'
UserValidity confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
EmailID/EmailNo  
Specify 'email'
EmailID/Email confines to:
  • Type is 'SCALAR'.
  • Datatype is 'EMAIL'.
Expirty Period typeYes  
Specify 'genexpiryperiodtype'
Expirty Period type confines to:
  • Type is 'SCALAR'.
  • Only '1', '2', '3' are allowed.
ValidityStartNo  
Specify 'validitystart'
ValidityStart confines to:
  • Type is 'SCALAR'.
  • Only 'Immediately', 'AfterFirstLogin' are allowed.
