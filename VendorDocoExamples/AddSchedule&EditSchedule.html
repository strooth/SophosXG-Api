Operation: Add Schedule / Edit Schedule
Description: To Add/Edit Schedule. Schedule defines a time schedule for applying firewall rules or Web and Application Filter policies. 

Sample Configuration
<Schedule> <Name>Name</Name> <Description>Text</Description> <Type>Recurring/OneTime</Type> <ScheduleDetails> <!-- for onetime --> <StartDate>2011-02-28 15:37:52</StartDate> <EndDate>2011-03-14 15:38:07</EndDate> <!-- for onetime ends --> <ScheduleDetail> <!-- for recurring type --> <Days>WeekDays</Days> <StartTime>00:00</StartTime> <StopTime>00:00</StopTime> <!-- for onetime --> <Days>WeekDays</Days> <StartTime>00:00</StartTime> <StopTime>00:00</StopTime> </ScheduleDetail> : </ScheduleDetails> </Schedule>



Parameter Mandatory Default Description
NameYes  
Specify name to identify the Schedule.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 50.
  • UTF-8 character(s) are allowed.
DescriptionNo  
Specify Schedule Description.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
TypeNo Recurring 
Select Type of Schedule: Recurring or One Time.
Type confines to:
  • Type is 'SCALAR'.
  • Only 'Recurring', 'OneTime' are allowed.
StartDateYes  
Specify Start date if 'One Time' Schedule type is selected.
StartDate confines to:

  • Datatype is 'STRING'.
EndDateYes
