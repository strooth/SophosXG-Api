Operation: Add Data Control List / Update Data Control List
Description:  

Sample Configuration
<MTADataControlList> <Name>Postal addresses</Name> <Signatures> <Signature>Postal addresses [Global]</Signature> <Signature>Postal addresses [Australia]</Signature> <Signature>Postal addresses [Canada]</Signature> <Signature>Postal addresses [Germany]</Signature> </Signatures> </MTADataControlList>



Parameter Mandatory Default Description
nameYes  
Specify 'name'
confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 255.
signaturenameNo  
Specify 'signaturename'
confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
