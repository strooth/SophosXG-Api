Operation: Add User Activity / Update User Activity
Description: To Add/Edit User activity for controlling user's Web Access. 

Sample Configuration
<UserActivity> <Name>Name</Name> <!-- for updating name --> <NewName>New Name</NewName> <Desc>Text</Desc> <CategoryList> <Category> <Type>web category/file type/url group</Type> <ID>Category name.</ID> </Category> : </CategoryList> </UserActivity>



Parameter Mandatory Default Description
NameYes  
Specify a name for the Web Filter Activity.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Characters not allowed: (^;'"\)
  • Maximum characters allowed are 50.
NewNameNo  
Change the activity name.
NewName confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Characters not allowed: (^;'"\)
  • Maximum characters allowed are 50.
DescriptionNo  
Specify Activity description.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 255.
CategoryListNo  
Specify categories the activity contains.
CategoryList confines to:
  • Type is 'ARRAY'.
  • Datatype is 'OBJECT'.
  • webfilter::useractivityrules
  • Multiple values are allowed.
