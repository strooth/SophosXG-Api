Operation: Add URL Group / Update URL Group / Add Individual URL
Description: To Add/Edit URL Group.Update URL groupAdd a URL to a group 

Sample Configuration
<WebFilterURLGroup> <Name>Name</Name> <URLlist> <URL>URLs</URL> <URL>URLs</URL> </URLlist> <Description>Text</Description> </WebFilterURLGroup>



Parameter Mandatory Default Description
NameYes  
Specify name for URL Group.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Comma (,) is not allowed.
  • Maximum characters allowed are 50.
  • UTF-8 character(s) are allowed.
DescriptionNo  
Specify Group description.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 255.
URLYes  
Specify the URLs to include in the Group.
URL confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 250.
  • Multiple values are allowed.
