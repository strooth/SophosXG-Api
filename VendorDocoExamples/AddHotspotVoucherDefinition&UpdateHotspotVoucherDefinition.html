Operation: Add Hotspot Voucher Definition / Update Hotspot Voucher Definition
Description:  

Sample Configuration
<VoucherDefinition> <Name>name</Name> <Description>description</Description> <ValidityPeriod>integer</ValidityPeriod> <ValidityUnit>Minutes/Hours/Days</ValidityUnit> <TimeQuota>integer</TimeQuota> <QuotaUnit>Minutes/Hours</QuotaUnit> <DataVolume>integer</DataVolume> <DataUnit>MB/GB</DataUnit> </VoucherDefinition>



Parameter Mandatory Default Description
NameYes  
Enter a descriptive name for the voucher definition.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
DescriptionNo  
Enter a description or other information.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
Validity PeriodYes  
Enter the time span for which a voucher with this definition will be valid.
Validity Period confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
Validity UnitNo  
Select the unit for Validity Period.
Validity Unit confines to:
  • Type is 'SCALAR'.
  • Only 'Minutes', 'Hours', 'Days' are allowed.
Time QuotaNo  
Enter the maximum online time after which a voucher of this definition expires.
Time Quota confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
Quota UnitNo  
Select the unit for Time Quota.
Quota Unit confines to:
  • Type is 'SCALAR'.
  • Only 'Minutes', 'Hours' are allowed.
Data VolumeNo  
Enter the maximum data volume to be transmitted with this voucher definition.
Data Volume confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
Data UnitNo  
Select the unit for Data Volume.
Data Unit confines to:
  • Type is 'SCALAR'.
  • Only 'MB', 'GB' are allowed.
