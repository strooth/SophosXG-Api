Operation: Add SSLVPN Bookmark Group / Edit SSLVPN Bookmark Group
Description: To Add/Edit SSL VPN Bookmark Group.To update SSL VPN Bookmark Group. 

Sample Configuration
<SSLBookmarkGroup> <Name>bookmarkgroupname</Name> <BookmarkList> <Bookmark>bookmark</Bookmark> : </BookmarkList> <Description>Text</Description> </SSLBookmarkGroup>



Parameter Mandatory Default Description
NameYes  
Specify a name for the bookmark Group.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 50.
BookmarkYes  
Select Bookmarks to be grouped.
Bookmark confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
  • Duplicate values will be ignored.
DescriptionNo  
Specify Bookmark Group description.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
